# Electronic Voting Machine (EVM)

This project is a simple Electronic Voting Machine (EVM) built using HTML, CSS, and JavaScript. It simulates the basic functionality of an electronic voting system where users can vote for their preferred candidate and view the results instantly.

## Features
- Vote for multiple candidates
- Live vote counting
- Prevents multiple votes (single vote per session)
- Responsive and user-friendly interface
- Clean and interactive UI
- Built using Vanilla JavaScript (No Frameworks)

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Learning Objectives
- DOM Manipulation
- Event Handling
- JavaScript Logic
- State Management
- Responsive Web Design
